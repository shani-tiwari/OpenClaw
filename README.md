# OpenClaw 🦅

> Your personal AI coding assistant that lives in your terminal and Telegram.

OpenClaw is an AI-powered developer assistant designed for developers who want a lightweight, extensible coding companion without relying on large IDE integrations.

It provides a CLI workflow for interacting with your projects and a Telegram interface for accessing the same assistant anywhere.

Powered by OpenRouter, OpenClaw can understand your codebase, plan new features, answer technical questions, manage development tasks, and safely perform file operations inside your workspace.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/shani-tiwari/OpenClaw.git
cd OpenClaw
```

Install dependencies

```bash
bun install
```

Start OpenClaw

```bash
openclaw wakeup
```

## 🔑 OpenRouter Setup

1. Create a free account on OpenRouter.
2. Generate an API key.
3. Add it to your environment variables.

```env
OPENROUTER_API_KEY=your_api_key
```

OpenClaw automatically uses supported free OpenRouter models.
